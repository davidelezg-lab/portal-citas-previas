# Definition of Done

Una tarea se considera terminada cuando:

- [ ] El código está subido en una rama descriptiva.
- [ ] Existe un Pull Request hacia main.
- [ ] El Pull Request está relacionado con un issue.
- [ ] Al menos un compañero ha revisado el cambio.
- [ ] La funcionalidad cumple los criterios de aceptación.
- [ ] No hay errores visibles en la aplicación.
- [ ] Se ha actualizado la documentación si procede.
- [ ] No se han incluido datos reales de ciudadanos.
